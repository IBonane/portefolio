    2019

    Projet académique "Camera-cooling": Développement d'un programme python permettant de compter des pixels chauds sur une image

    Travail en groupe

    Belfort, France

    Le projet portait sur le refroidissement d'une petite camera d'astrophoto. Dans un premier, nous avions voulu déterminer l'effet du temps de pause sur la qualité de l'image, obtenues à des intervalles de temps donnée. Pour ce faire,

    Nous sommes passés à l'analyse de ces dernières via notre "programme Python" (en vue de voir le nombre aproximatif de pixels chauds sur chaque image en fonction du temps).

    Ce qui nous a permis, dans un second temps, de concevoir notre dispositif de refroidissement de manière optimale.
    Voir le code source sur Github
