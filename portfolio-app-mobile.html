Application mobile

j'ai réalisé ce projet pour rigoler avec les amis.

L'objectif du projet est de permettre d'indentifier toute les personnes qu'il prend en photo.

En effet, une fois une photo prise, elle est envoyer sur le serveur de Microsoft Azure Cognitive (utilisant un API IA) pour une reconnaissance faciale. Après analyse, nous obtenons via l'interface de l'application, le genre et l'âge aproximatif de la personne.

Technologies utilisées : C#, Xamarin-forms, JSON
